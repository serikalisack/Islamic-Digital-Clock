# Islamic Calendar Feature

## ⚠️ IMPORTANT NOTICE

This Islamic calendar feature provides **astronomical calculations only**. Islamic dates, especially Eid dates, are determined by **moon sighting** and **local Islamic authority rulings**, not astronomical calculations alone.

**Always verify dates with your local mosque or Islamic authority before acting on them.**
