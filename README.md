# Islamic-Digital-Clock
An elegant Islamic digital clock with real-time updates, calendar toggles, and greeting messages based on time of day.

## ⚠️ Islamic Calendar Notice

**This application provides astronomical calculations for Islamic dates. Actual Islamic dates, especially Eid dates, are determined by moon sighting and local Islamic authority rulings.**

**Always verify important dates with your local mosque or Islamic authority. Do not rely solely on this application's calculations for religious observances.**

See [ISLAMIC_CALENDAR_README.md](ISLAMIC_CALENDAR_README.md) for detailed information about Islamic date calculations.

## Features

- Real-time Islamic clock
- Hijri and Gregorian calendar display
- Prayer times calculation
- Qibla direction compass
- Islamic inspiration (verses and hadiths)
- Eid date estimation
- Zakat calculator
- Ramadan countdown
- Multi-language support
- Dark mode
- PWA (Progressive Web App)

## Installation

1. Clone the repository
2. Open `index.html` in a modern web browser
3. For PWA installation, use the browser's "Add to Home Screen" feature

## Usage

- The app works offline after initial load
- Location permission is required for accurate prayer times and Qibla direction
- Notification permission enables prayer time alerts

## Technologies

- HTML5, CSS3, JavaScript (ES6+)
- Moment.js and Moment-Hijri for date calculations
- Aladhan API for prayer times
- Device Orientation API for compass
- Service Workers for PWA functionality
